# LED-Blinker-
I will be making my first hardware project, LED blinker. I made this because I wante to deisgn a cool project as my first and dive into hardware. I really like how it turned out!

## Parts 
555 IC
4017 IC
Header pins
electrolytic capacitor
1k ohm resistor
470 ohm resistor
potentiometer
10 leds
